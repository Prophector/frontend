About Prophector

Prophector is a web application that lets you analyze COVID-19 data and create forecasts.

Data Sources

The sources used are Our World in Data and the Federal Office of Public Health.

Data is scraped daily and filled into our database automatically.

Predictions

We use Prophet to create forecasting models. If you want to learn more how it works, check out their documentation. These are our favorites:

  • Diagnostics
  • Trend Changepoints
  • Seasonality, Holiday Effects, And Regressors

Source Code

The source code for this application can be found at github.com/Prophector.
